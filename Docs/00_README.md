# SillyTavern Solo TTRPG System

A design and build bundle for running endless solo tabletop RPG campaigns in SillyTavern, with:

- Pre-authored adventures generated by a Python pipeline (you play blind)
- A pack system that cleanly separates genre (Symbaroum, cyberpunk, space opera, cosmic horror) from engine and campaign
- A custom SillyTavern extension for character sheets, dice rolls, status tracking, canon detection, and backup/restore
- A PbtA-inspired 2d6 + attribute resolution system with six attributes (fixed)
- Character abilities that fit the genre (magic, cyberware, psionics, etc.)

## Files in this bundle

1. **`01_SYSTEM_OVERVIEW.md`** — architecture, components, how content flows. Start here.
2. **`02_GENRE_PACK_SPEC.md`** — the pack contract every genre pack must satisfy.
3. **`03_CAMPAIGN_GENERATOR_BRIEF.md`** — Claude Code brief for the Python campaign generator.
4. **`04_EXTENSION_BRIEF.md`** — Claude Code brief for the SillyTavern extension.
5. **`05_PACK_GENERATOR_BRIEF.md`** — Claude Code brief for the Python pack generator.
6. **`06_PACK_AUTHORING_GUIDE.md`** — reference for humans (or LLMs) writing packs by hand.
7. **`07_GM_BASE_PROMPT.md`** — the engine-level GM system prompt.
8. **`08_EXAMPLE_PACK_SYMBAROUM.md`** — a complete, illustrative Symbaroum-flavored dark fantasy pack.

## Recommended reading order

- Everyone reads 01 first.
- Then 02 (the pack contract).
- Then pick your path:
  - Playing a campaign right away with minimal tooling: 07 + 08
  - Building the extension first: 04 + 07
  - Building the campaign generator first: 03 + 02
  - Building the pack generator: 05 + 06

## Recommended build order

Don't build everything upfront. Priority:

1. **Extension tiers 1.1 and 1.2** (character sheet, dice rolling) — minimum viable automation
2. **A hand-written Symbaroum pack** — use file 08 as the starting point
3. **Campaign generator** — once you know what you want authored vs. improvised
4. **Extension tiers 2 and 3** — as pain points emerge in play
5. **Pack generator** — when you want a second genre

This gets you playing in days, and every piece of tooling is built against real friction.
